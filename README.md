# Description : manage_book 

c'est un projet d'API pour la gestion des livres et apprendre FastAPI

## Installation

- **Avec Github :**
  ```bash
  git clone https://github.com/elielabah/manage_book/
  ```

## Utilisation

-  **Création d'un environnement virtuel :**

```cmd
  python -m venv env
```

-  **Installation des dépendances :**
```bash
  pip install -r requirements.txt
```

-  **Initialiser alembic :**

```cmd
  alembic init alembic
```

- Configuer alembic :

    - Modifier la ligne sqlalchemy.url

    - Modifier alembic\env.py et importer les metadata

- **Lancement du serveur :**

```python
  python __main__.py
```

## `Technologies et bibliothèques`

- Python
- Pydantic
- FASTAPI
- SQLAlchemy
